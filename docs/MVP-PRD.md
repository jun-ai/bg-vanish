## 1. 项目背景与目标

### 1.1 背景
"Image Background Remover" 是海外搜索量极高的长尾关键词（月搜索量 50 万+），现有工具如 remove.bg、clipdrop 等功能强大但价格较高，市场存在对免费/低价、操作简单、响应快速的工具的需求。

### 1.2 目标
- **首要目标**: 上线 MVP，通过 SEO 自然流量获取用户，验证市场需求
- **次要目标**: 积累用户量，为后续付费转化（API 服务、垂直场景）打基础
- **关键指标**:
  - 日均 UV ≥ 500（上线 3 个月内）
  - 用户留存率（7 日回访）≥ 15%
  - 单次处理成功转化率 ≥ 90%

## 3. 功能需求

### 3.1 P0 — MVP 必备（v1.0）

#### F-01 图片上传
- 支持拖拽上传、点击选择文件、键盘粘贴（Ctrl+V）
- 支持格式：PNG、JPG、WebP
- 单文件大小限制：≤ 10MB
- 单次批量上传：≤ 5 张
- 上传前本地预览

#### F-02 AI 背景消除
- 一键去除背景，调用 Remove.bg API
- 返回透明背景 PNG
- 并发处理：最多 2 张同时处理
- 单张处理超时：30 秒
- 处理中显示加载动画

#### F-03 结果预览
- 原图 vs 结果图左右对比展示
- 透明背景用棋盘格表示
- 支持缩放查看细节

#### F-04 背景替换
- 预设背景色：透明、白色、黑色、红/蓝/绿
- 自定义颜色（取色器）
- 背景替换仅在预览层生效，下载时可选择是否应用

#### F-05 下载
- 下载透明背景 PNG
- 下载带背景色 PNG
- 文件名自动加上 `-no-bg` 后缀

#### F-06 免费额度控制
- 免费：5 张/天（基于 localStorage）
- 用完后弹窗提示"今日额度已用完"
- 显示剩余额度（右上角）

## 4. 非功能需求

### 4.1 性能
| 指标 | 目标 |
|------|------|
| 页面首屏加载 | ≤ 2s（全球 P95） |
| 图片上传到开始处理 | ≤ 1s |
| 背景消除结果返回 | ≤ 10s（中等尺寸图片） |
| 并发支持 | ≥ 50 RPS（Workers 免费额度内） |

### 4.2 安全
- API Key 存储在 Cloudflare Worker Secrets，不暴露给前端
- 所有处理在服务端完成，不存储用户图片
- 不记录用户个人信息（MVP 阶段无登录）
- 输入校验：文件类型、大小、频率限制

### 4.3 可用性
- 部署在 Cloudflare Workers，全球边缘节点
- 目标可用性：99.5%+
- Remove.bg API 故障时返回友好错误提示

### 4.4 兼容性
- 浏览器：Chrome 90+、Firefox 90+、Safari 15+、Edge 90+
- 移动端：iOS Safari、Android Chrome
- 屏幕尺寸：320px ~ 2560px 响应式

## 5. 技术架构

```
┌─────────────┐     ┌──────────────────────┐     ┌───────────────┐
│   用户浏览器  │────▶│  Cloudflare Workers   │────▶│ Remove.bg API │
│  (前端 HTML)  │◀────│  (后端代理 + 内嵌HTML) │◀────│  (AI 推理)     │
└─────────────┘     └──────────────────────┘     └───────────────┘
                           │
                    ┌──────┴──────┐
                    │  无存储层    │
                    │  纯内存处理  │
                    └─────────────┘
```

### 技术选型

| 层级 | 技术 | 理由 |
|------|------|------|
| 前端 | HTML + Tailwind CSS (CDN) | 零构建、快速迭代、体积小 |
| 后端 | Cloudflare Workers (TypeScript) | 免费额度大、全球边缘、按请求计费 |
| AI 接口 | Remove.bg API | 效果业界顶尖、简单易集成 |
| 存储 | 无 | 符合"不存储用户数据"的隐私定位 |
| 部署 | Wrangler CLI | Cloudflare 官方工具 |

## 6. 数据流

```
用户选择图片
    ↓
前端校验（格式、大小、额度）
    ↓
FormData → POST /api/remove-bg
    ↓
Worker 校验 → 转发 Remove.bg API
    ↓
Remove.bg 返回 PNG → Worker 返回给前端
    ↓
前端展示结果 → 用户选择背景色 → 下载
```

| 风险 | 影响 | 概率 | 应对策略 |

## 12. 附录

### A. Remove.bg API 参考
- **Endpoint**: `POST https://api.remove.bg/v1.0/removebg`
- **认证**: `X-Api-Key` header
- **参数**: `image_file` (binary), `size` (auto | preview | full)
- **返回**: PNG binary (Content-Type: image/png)
- **错误码**: 400 (参数错误), 402 (额度不足), 429 (限流), 500 (服务错误)

### B. Cloudflare Workers 限制
- 免费版: 10ms CPU time/请求, 100k 请求/天
- Paid ($5/月): 50ms CPU time, 1000 万请求/月
- 注意: `image_file` 透传不消耗 Worker CPU 时间，主要等待 Remove.bg 响应
